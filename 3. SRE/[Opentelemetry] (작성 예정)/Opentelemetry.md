### Opentelemetry (Otel)

* Traces, Metrics, Logs 같은 데이터를 instrumenting, generating, collecting, exporting 할 수 있는 Observability 프레임 워크
* 데이터를 저장하거나 쿼리하는 방법을 제공하지 않고, 오픈 소스 백엔드(Jaeger, Prometheus, Grafana)로 데이터를 전송하여 데이터를 저장한다.
